# 📖 our story — handmade aesthetic scrapbook album

repositori kliping memori digital interaktif dengan estetika pinterest romantis, pencahayaan hangat, tekstur berlapis, dan musik latar kustom.

## 🛠️ cara pakai (terima beres):
1. buat sebuah repositori baru di github kamu.
2. unggah semua file ini (`index.html`, `style.css`, `script.js`) langsung ke repositori tersebut.
3. buat struktur folder bernama `assets/images/` di dalam repositori kamu.
4. ekstrak isi file `photo.zip` kamu dan ubah nama total 10 foto tersebut menjadi:
   - `photo1.jpg` (untuk sampul depan)
   - `photo2.jpg` sampai `photo9.jpg` (halaman isi cerita)
   - `photo10.jpg` (untuk sampul belakang)
5. letakkan 10 foto tersebut ke dalam folder `assets/images/`.
6. masukkan file `lagu.mp3` kamu langsung di folder utama repositori (sejajar dengan `index.html`).

## 🚀 mengaktifkan link website:
1. masuk ke tab **settings** di repositori github kamu.
2. cari menu **pages** di bilah sisi kiri.
3. pada bagian **build and deployment**, ubah branch menjadi `main` atau `master`, lalu klik **save**.
4. tunggu sekitar 1-2 menit, github akan memberikan tautan live website kliping romantis kamu!
